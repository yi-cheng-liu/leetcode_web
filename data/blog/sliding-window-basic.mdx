<summary>1248. Count Number of Nice Subarrays</summary>
<blockquote>Find the first array with k odd numbers, then move i as well to maintain k. </blockquote>
<div class="flex flex-col md:flex-row gap-4">
  <div class="md:w-2/5">
    <div>
        Given an array of integers nums and an integer k. A continuous subarray is called nice if there are k odd numbers on it.

        Return the number of nice sub-arrays.
    </div>
    
    <blockquote>
        Input: nums = [1,1,2,1,1], k = 3<br />
        Output: 2<br />
        Explanation: The only sub-arrays with 3 odd numbers are [1,1,2,1] and [1,2,1,1].
    </blockquote>
  </div>
  <div class="md:w-3/5">
    ```c
    class Solution {
    public:
        int numberOfSubarrays(vector<int>& nums, int k) {
            int ans = 0, i = 0, count = 0, n = nums.size();
            for (int j = 0; j < n; j++) {
                // Identify the odd number
                if (nums[j] & 1) {
                    k--;
                    count = 0;
                }
                while (k == 0) {
                    k += nums[i] & 1;
                    i++;
                    count++;
                }
                ans += count;
            }

            return ans;
        }
    };
    ```
  </div>
</div>

<summary>1358. Number of Substrings Containing All Three Characters</summary>
<blockquote>Use an array then record the 3 characters, then move i as well to maintain all 3. </blockquote>
<div class="flex flex-col lg:flex-row gap-4">
  <div class="lg:w-2/5">
    <div>
        Given a string s consisting only of characters a, b and c.

        Return the number of substrings containing at least one occurrence of all these characters a, b and c.
    </div>
    
    <blockquote>
        Input: s = "abcabc"<br />
        Output: 10<br />
        Explanation: The substrings containing at least one occurrence of the characters a, b and c are "abc", "abca", "abcab", "abcabc", "bca", "bcab", "bcabc", "cab", "cabc" and "abc" (again). 
    </blockquote>
  </div>
  <div class="lg:w-3/5">
    ```c
    class Solution {
    public:
        int numberOfSubstrings(string s) {
            // count of a, b, c
            int count[3] = {0, 0, 0};
            int n = s.length();
            int ans = 0;
            int i = 0;
            for (int j = 0; j < n; j++) {
                count[s[j] - 'a']++;
                while (count[0] && count[1] && count[2]) {
                    count[s[i] - 'a']--;
                    i++;
                }
                ans += i;
            }

            return ans;
        }
    };
    ```
  </div>
</div>
